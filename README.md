# PokeSync - Game Service

## Building and Running the Application

From the root directory of the project:

```
sh start.sh
```

### With Docker

```
docker build -t game-service .
docker run -t game-service .
```

Docker-Compose is also supported:

```
docker-compose up
docker-compose down
```
